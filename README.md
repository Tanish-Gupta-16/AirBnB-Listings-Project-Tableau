# AirBnB-Listings-Project-Tableau
This project analyzes Airbnb listings data from Seattle to uncover pricing patterns, seasonal demand trends, and market opportunities for prospective property investors and hosts.
Using Tableau, multiple datasets were combined and transformed to create an interactive dashboard that provides insights into revenue potential, location-based pricing, property characteristics, and competitive dynamics within the Airbnb marketplace.
# Key Insights and Features

• Analyzed 3,600+ Airbnb listings across Seattle neighborhoods and zip codes.

• Identified high-value locations based on average listing prices.

• Explored seasonal pricing trends using daily calendar data.

• Examined the relationship between bedroom count and average price.

• Evaluated competition levels through listing distribution by property size.

• Enabled location-based analysis through interactive geographic visualizations.
